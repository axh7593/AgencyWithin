About this Project

I began the project by finding a good bootstrap template to showcase the project on. I wanted something that was not to much but still looked good in terms of design and ux/ui. I believe that using bootstrap for this project was a good idea because that way I could focus more on implementation of features and have more time to find and understand an API.
Speaking of the API, I began by trying to use the one suggested in the project documentation, openweathermap.org but found myself not liking it, so I began in search of another API that would show weather and I landed upon using https://darksky.net/ which is another widely used weather API.
I began to look at documentation and online resources on how to use the API and how to begin implementation. I began coding and after a couple hours relized that I would need to streamline the process of making the API calls as dark sky requires for each call a set of latitude and longitude coords everytime a call is sent. So I created a function with several paramters (createCall) that would be used in my main HTML file to pass in the required info that I needed (this can be found in js/app.js)
